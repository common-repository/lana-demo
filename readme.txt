Demo user with editable roles

== Description ==

Demo user with editable roles and dashboard widgets.

= Role options: =
* upload files
* edit posts
* edit pages

You can add two dashboard widget.

= Lana Codes =
[Lana Demo](http://lana.codes/lana-product/lana-demo/)

== Installation ==

= Requires =
* WordPress at least 4.0
* PHP at least 5.3

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/lana-demo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

= How to use it =
* in `Settings > Lana Demo`, you can set the demo username and password.
